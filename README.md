## 使用说明

  * 用了jsp技术，所以需要tomcat等能承载java程序的容器，放到tomcat的webapps目录下就可以了
  * 使用了bootstrap框架做了响应式的分辨率处理，支持iPhone、iPad等移动设备访问
  * 主页是index.jsp
